# Heart Disease Prediction using Machine Learning

## Overview

This project predicts whether a patient has heart disease using machine learning classification models.

## Models Used

* Logistic Regression
* Decision Tree
* Random Forest

## Dataset

* 303 patient records
* 14 medical features

## Features

* age
* cp
* chol
* thalach
* oldpeak
* ca
* and others

## Results

* Logistic Regression Accuracy: ~85%
* Decision Tree Accuracy: ~75%
* Random Forest gave best performance

## Feature Importance

Important predictors:

* oldpeak
* thalach
* ca
* cp
